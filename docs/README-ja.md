# Judgelight
### Judgelight is the judgement server for competitive-programming.
![logo](/docs/img/logo.png)

## Features
- ローカルで競技プログラミングが実現できる
  - 問題を外部に公開することなくコンテストが実施可能
  - 教育現場や内輪での実施に最適
- 手軽な問題管理
  - YAMLによる問題管理
- 手軽な導入方法
  - Dockerコンテナによる環境構築の効率化
  - 必要な環境はDockerとGoだけ
